# wgpu_minecraft

Rustとwgpuを使用して作成中の自作Minecraft。まだ開発途中です。

## 前提条件

- Rustのインストールが必要です。

## 使用方法

1. リポジトリをクローンします。
```bash
git clone https://github.com/yourusername/wgpu_minecraft.git
```
2. ディレクトリに移動します。
```bash
cd wgpu_minecraft
```
3. 実行
```bash
cargo run 
```

## 進行状況

現在、マップの生成までを実装しています。次のステップとして、マウスの座標計算とブロックの追加・削除の機能を実装予定です。
